stdin	0	Стандартный поток ввода
stdout	1	Стандартный поток вывода
stderr	2	Стандартный поток вывода ошибок

> используется для перенаправления вывода в файл

>> похожа на > с тем отличием, что оператор >> не удаляет содержимое файла, а дописывает новые данные к уже существующим.

2> перенаправляет стандартный вывод ошибок — standard error (stderr)

2>&1 предназначена для перенаправления стандартного вывода и стандартного вывода ошибок в один поток, 
который затем может быть перенаправлен в единый файл

&> для перенаправления как стандартного вывода, так и стандартной ошибки в файл

Итак, &> является аналогом 2>&1, а &>> это то же самое, но с перенаправлением вывода в файл

<, то он перенаправляет ввод

<< называется here document. С его помощью можно ввести строку состоящую из нескольких строк или присвоить переменной многострочное значение.

cat > file.txt << "EOF"
1
2
3
EOF




Сочетания клавиш
Сочетание	Описание
CTRL-A	Перемещение курсора в начало строки
CTRL-E	Перемещение курсора в конец строки
CTRL-R	Поиск по истории
CTRL-W	Вырезать последнее слово
CTRL-U	Вырезать всё до курсора
CTRL-K	Вырезать всё после курсора
CTRL-Y	Вернуть последнюю вырезанную строку
CTRL-_	Отмена
CTRL-L	Очистка экрана терминала
